# Finance Bot — Roadmap

## ✅ MVP (реализовано)
- Добавление транзакций (тип → категория → сумма)
- 15 категорий с эмодзи (доходы + расходы)
- Статистика за день / неделю / месяц с прогресс-барами
- История последних 10 транзакций
- Удаление последней транзакции
- Баланс за текущий месяц на главном экране
- Multi-user (SQLite, нормализованная схема, ACID/WAL)
- Валюта MDL (L)

## 💰 Монетизация

| Модель | Детали |
|--------|--------|
| **Freemium** | Бесплатно: 1 кошелёк, 50 транзакций/мес · Платно: без лимитов + отчёты |
| **Подписка** | ~1–2 $/мес за расширенный функционал |
| **B2B** | Учёт расходов для малого бизнеса / команды |

---

## Приоритет реализации

```
1. Заметка к транзакции    ← быстро, сразу поднимает качество
2. Лимиты по категориям    ← ключевая польза для пользователя
3. Автоотчёт               ← retention
4. Экспорт CSV             ← доверие и "взрослость" продукта
5. ИИ-анализ               ← дифференциатор при питче
```
